

## הסבר תהליך ההזמנה + תוכנית לשיפור

### איך התהליך עובד היום

```text
בעלים (Owner)                        מוזמן (Co-parent)
────────────────                     ──────────────────
1. מזין אימייל בטופס ←──────────────→ (עדיין לא יודע)
2. invite_co_parent() יוצר שורה
   ב-family_members עם:
   - status = 'pending'
   - invited_email = 'x@y.com'
   - id = UUID (זה ה-inviteId)
3. צריך לשלוח לינק ידנית ←──────────→ 4. מקבל לינק:
                                         /accept-invite/{UUID}
                                      5. אם לא מחובר → /auth
                                      6. אחרי לוגין → חוזר ללינק
                                      7. לוחץ "קבל הזמנה"
                                      8. accept_family_invite() בודק:
                                         - סטטוס = pending? ✓
                                         - אימייל תואם? ✓
                                         - לא הבעלים עצמו? ✓
                                         - יש שורת parents? ✓
                                      9. מעדכן status → accepted
                                         member_id → auth.uid()
```

### אבטחת הלינק — כבר מובנית

הלינק **לא פתוח לכולם**. שלוש שכבות הגנה כבר קיימות:

1. **UUID אקראי** — הלינק מכיל UUID שלא ניתן לנחש (`/accept-invite/3f8a...`). אי אפשר לסרוק או לנחש אותו.

2. **בדיקת אימייל** — ה-RPC `accept_family_invite` משווה את האימייל של המשתמש המחובר ל-`invited_email` שהבעלים הזין. אם האימייל לא תואם → `EMAIL_MISMATCH` → נדחה.

3. **שימוש חד-פעמי** — ברגע שמישהו מקבל את ההזמנה, הסטטוס משתנה ל-`accepted`. ניסיון נוסף עם אותו לינק → `INVITE_NOT_FOUND` (כי הוא מחפש `status = 'pending'` בלבד). הלינק בפועל "משמיד את עצמו".

**לסיכום: הלינק בטוח — רק מי שיודע את ה-UUID וגם רשום עם האימייל הנכון יכול לקבל את ההזמנה, ורק פעם אחת.**

### מה חסר — כפתור שיתוף בוואטסאפ

כרגע הבעלים צריך להעתיק את הלינק ידנית. צריך להוסיף כפתור "שלח בוואטסאפ" שפותח את WhatsApp עם הודעה מוכנה.

### תוכנית שינויים

**קובץ אחד: `src/pages/FamilyV2.tsx`**

בקטע של "ממתין לאישור" (pending state, שורות 488-523), להוסיף:

1. **שורת לינק להעתקה** — מציגה את ה-URL המלא עם כפתור העתקה (clipboard)
2. **כפתור "שלח בוואטסאפ"** — פותח `https://wa.me/?text=...` עם הודעה בעברית שכוללת את הלינק, למשל:
   > הי! הזמנתי אותך להצטרף כהורה שותף באפליקציית Kippy. לחץ/י על הלינק: {URL}

שני הכפתורים יופיעו רק כשיש הזמנה פתוחה (status = pending).

**אין שינויי backend** — הלינק כבר מאובטח ברמת ה-RPC.
