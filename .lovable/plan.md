

# הפיכת כרטיסי KPI בדשבורד ניהול ללחיצים

## מה ישתנה

כל כרטיסי הסטטיסטיקה (KPI) בטאב "סקירה כללית" יהפכו ללחיצים. לחיצה על כרטיס תעביר אותך ישירות לטאב הרלוונטי עם הנתונים המפורטים. הכרטיסים יקבלו אפקט hover (צל + סמן יד) כדי שיהיה ברור שהם אינטראקטיביים.

## מיפוי כרטיסים לטאבים

| כרטיס | לחיצה מעבירה ל... |
|--------|-------------------|
| ילדים פעילים היום | טאב משתמשים (עם סינון "פעיל היום") |
| הורים פעילים השבוע | טאב משתמשים |
| הודעות שנסרקו היום | טאב AI Insights |
| התראות שנשלחו היום | טאב Training |
| אחוז התראות עם משוב | נשאר בסקירה (כבר מוצג כאן) |
| הורים רשומים | טאב משתמשים |
| רשימת המתנה | טאב רשימת המתנה |
| מכשירים מחוברים | טאב משתמשים (עם סינון "אונליין") |
| התראות היום | טאב Training |
| התראות קריטיות | טאב Training |
| פעילים היום | טאב משתמשים (עם סינון "אונליין") |
| שלבי משפך ההמרה | לחיצה על כל שלב תעביר לטאב הרלוונטי |

## שינויים טכניים

### 1. `src/pages/Admin.tsx`
- הוספת state חדש: `usersStatusFilter` לסינון ראשוני בטאב משתמשים
- העברת callback `onNavigate(tab, filter?)` ל-AdminOverview
- כשהמשתמש לוחץ כרטיס, מתבצע `setActiveTab(tab)` ואם יש סינון - מעדכן את ה-filter

### 2. `src/pages/admin/AdminOverview.tsx`
- קבלת prop חדש: `onNavigate: (tab: string, filter?: string) => void`
- עטיפת כל Card ב-`onClick` עם `cursor-pointer` ו-`hover:shadow-lg hover:border-primary/40 transition-all`
- הוספת אייקון חץ קטן (ChevronLeft כי RTL) בפינת כל כרטיס לחיץ

### 3. `src/pages/admin/AdminUsers.tsx`
- קבלת prop חדש: `initialStatusFilter?: string`
- אם מגיע filter מבחוץ, להחיל אותו על ה-Select של סטטוס

### סגנון
- כרטיסים לחיצים: `cursor-pointer hover:shadow-md hover:border-primary/40 transition-all duration-200`
- אייקון חץ קטן בכל כרטיס לחיץ
- ללא שינוי בכרטיסים שאינם לחיצים (גרפים)
