## הבהרת הניסוח בכרטיס "התקינו את KippyAI כאפליקציה"

הניסוח הנוכחי בכרטיס ההתקנה (`src/components/InstallAppCard.tsx`) עלול להטעות הורים לחשוב שמדובר באפליקציה שמותקנת על מכשיר הילד (לצורך הניטור), בעוד שהכוונה היא לקיצור של **לוח ההורה** עצמו למסך הבית של ההורה — אותו אפשר לפתוח מכל טלפון בלי להוריד דבר מחנות האפליקציות.

### שינויים (קובץ אחד: `src/components/InstallAppCard.tsx`)

**1. וריאנט `landing` (שורות 82–83):**
- כותרת: `התקינו את KippyAI כאפליקציה` → **`קיצור ללוח ההורה במסך הבית`**
- תיאור: `גישה מהירה ממסך הבית — בלי חנות אפליקציות, עובד בכל טלפון` → **`לוח ההורה שלכם, גישה מהירה ממסך הבית של הטלפון. לא מחליף את אפליקציית הילד — זו אפליקציה נפרדת בשבילכם, ללא צורך בחנות אפליקציות.`**

**2. וריאנט `settings` (שורות 100–101):**
- כותרת: `התקינו את Kippy כאפליקציה` → **`קיצור ללוח ההורה במסך הבית`**
- תיאור: `גישה מהירה ממסך הבית, חוויית אפליקציה מלאה` → **`פתיחה מהירה של לוח ההורה מהטלפון, בלי חנות אפליקציות`**

**3. הוראות פנימיות (שורות 34, 48, 62–68):**
- iOS (שורה 34): `להתקנה ב-iPhone:` → **`להוספת לוח ההורה למסך הבית ב-iPhone:`**
- Android (שורה 48): `להתקנה ב-Android:` → **`להוספת לוח ההורה למסך הבית ב-Android:`**
- Desktop (שורות 62–68):
  - כותרת: `רוצים להתקין כאפליקציה?` → **`רוצים לפתוח את לוח ההורה גם מהטלפון?`**
  - גוף: `פתחו את האתר בטלפון (Chrome ב-Android, או Safari ב-iPhone) ותקבלו אפשרות התקנה ישירה למסך הבית — ללא חנות אפליקציות.` → **`זו האפליקציה שלכם כהורים (לא של הילד). פתחו את הכתובת בטלפון (Chrome ב-Android, או Safari ב-iPhone) ותקבלו אפשרות להוסיף קיצור למסך הבית — בלי להוריד אפליקציה מהחנות.`**

**4. כפתור ההתקנה (שורה 30):**
- `התקנה` → **`הוסיפו למסך הבית`** (תיאור מדויק יותר ל-PWA install ולא מתבלבל עם "התקנת אפליקציה" מהחנות).

אין שינוי בלוגיקת ההתקנה (`usePWAInstall`), בעיצוב, או באייקונים.
