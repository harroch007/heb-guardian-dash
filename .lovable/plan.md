

# אחידות ויזואלית בכרטיסיות במצב סגור

## הבעיה
כרגע כל כרטיסייה מציגה מידע שונה במצב סגור:
- **זמן מסך** — מציג את הזמן (0:03) ליד הכותרת
- **לוחות זמנים** — מציג "3 פעילים"
- **ניהול אפליקציות** — מציג badges של "חסומות" ו"חדשות"
- **מיקום** — מציג כתובת
- **גדר גיאוגרפית** ו**משימות ובונוס** — כבר נקיים (Accordion)

המשתמש רוצה שכל הכרטיסיות יהיו אחידות: רק שם + אייקון, אותו גופן ומשקל, ללא נתונים במצב סגור.

## שינויים

### 1. `ScreenTimeSection.tsx` — הסרת תצוגת זמן במצב סגור
- הסרת ה-`<span>` שמציג `formatScreenTime(currentUsageMinutes)` מה-CardHeader (שורות 103-106)
- השארת רק אייקון + "זמן מסך" + חץ

### 2. `SchedulesSection.tsx` — הסרת "X פעילים" במצב סגור
- הסרת הבלוק `{!expanded && activeCount > 0 && ...}` (שורות 84-88)
- השארת רק אייקון + "לוחות זמנים" + חץ

### 3. `AppsSection.tsx` — הסרת badges במצב סגור
- הסרת שני הבלוקים `{!expanded && blockedTotal > 0 && ...}` ו-`{!expanded && pendingApps.length > 0 && ...}` (שורות 117-126)
- השארת רק אייקון + "ניהול אפליקציות" + חץ

### 4. `LocationSectionV2.tsx` — הסרת כתובת במצב סגור
- הסרת הבלוק `{!expanded && hasLocation && ...}` (שורות 79-83)
- השארת רק אייקון + "מיקום" + חץ

### 5. אחידות גופן וגודל בכל הכותרות
- וידוא שכל ה-CardTitle משתמש ב-`text-sm font-semibold` (במקום `text-base`) כדי להתאים לסגנון של GeofenceSection ומשימות ובונוס
- האייקונים: `w-5 h-5 text-primary` בכולם

## תוצאה
כל 6 הכרטיסיות ייראו זהות במצב סגור: אייקון + שם בלבד, באותו גודל ומשקל גופן.
